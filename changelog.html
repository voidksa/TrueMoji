v2.0.1

What's new

TrueMoji 2.0.1 focuses on NotebookLM support, cleaner text fields, and better coverage on dynamic websites.

  • 📓

    NotebookLM support

    TrueMoji now works better inside NotebookLM quizzes, flashcards, and delayed Shadow DOM areas.

  • ✍

    Cleaner text fields

    Editable fields now keep native input behavior, preventing duplicate emojis and overlapping text.

  • 🌐

    Better dynamic page coverage

    TrueMoji watches late text updates more reliably and uses a fallback path for strict website image policies.

Support continued updates

TrueMoji is maintained in spare development time. Your support helps keep compatibility fixes and new emoji improvements moving.
